Projet d'apprentissage Django

Cette application web a été développée dans le cadre d'un projet d'apprentissage du framework Django. Elle permet la gestion de groupes de musique et leurs annonces associées.

Technologies utilisées :

  • Python/Django
  • Bootstrap 5

Fonctionnalités :

  • Gestion des groupes (CRUD)
  • Gestion des annonces (CRUD) - Relation ManyToOne vers Groupes
  • Interface responsive
  • Formulaire de contact
